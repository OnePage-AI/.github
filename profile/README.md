## 프로젝트 소개

원페이지는 방대한 강의 자료를 AI 기술로 구조화하고 재구성하여, 사용자가 핵심 내용을 빠르게 습득하고 효과적으로 복습하며 시험을 대비할 수 있도록 돕는 학습 보조 서비스입니다.
단순한 자료 요약을 넘어, 실제 강의 흐름과 시험에 중요한 내용 위주로 정보를 선별하고 정리합니다.

## 문제 정의

많은 학습자가 다음과 같은 문제에 직면합니다.

*   **비효율적인 자료 정리:** 방대한 강의 자료를 직접 정리하는 데 많은 시간을 할애하여, 정작 내용을 이해하고 복습할 시간이 부족합니다.
*   **낮은 학습 효과:** 단순히 자료를 읽는 것만으로는 학습 효과가 떨어지며, 스스로 핵심 내용을 선별하고 구조화하는 데 어려움을 겪습니다.
*   **부족한 시험 대비:** 시험 대비를 위한 효과적인 요약 자료나 자가 진단 퀴즈를 직접 만드는 것이 번거롭고 시간이 많이 소요됩니다.

## 핵심 기능

원페이지는 다음 핵심 기능을 통해 사용자의 학습 과정을 지원합니다.

*   **다양한 자료 형식 지원:** PDF, PPTX, DOCX 등 여러 형식의 강의 자료를 웹 페이지에 직접 업로드할 수 있습니다.
*   **AI 기반 강의 내용 분석:** 업로드된 자료의 텍스트를 AI가 분석하여 핵심 내용을 추출하고, 강의 흐름에 맞춰 체계적으로 구조화합니다.
*   **맞춤형 학습 결과물 생성:**
    *   **강의 복습 모드:** 강의 내용을 깊이 있게 이해할 수 있도록 체계적으로 정리된 학습 노트를 생성합니다.
    *   **핵심 요약 모드:** 시험 직전 빠르게 핵심 내용을 파악하고 암기할 수 있도록 간결하게 요약된 자료를 제공합니다.
    *   **퀴즈 생성 모드:** 강의 내용을 바탕으로 객관식/주관식/단답형 문제와 상세한 해설을 생성하여 학습 효과를 검증하고 취약점을 파악합니다.
*   **학습 기록 관리:** 사용자의 모든 강의 자료 및 AI 분석 결과물을 개인 보관함에 안전하게 저장하고, 언제든지 다시 열람할 수 있도록 관리합니다.

## 서비스 사용 흐름

간단한 몇 단계만으로 원페이지의 강력한 학습 기능을 경험할 수 있습니다.

1.  **로그인:** Google 계정을 통해 원페이지 서비스에 간편하게 로그인합니다.
2.  **강의 자료 업로드:** 분석을 원하는 PDF, PPTX, DOCX 형식의 강의 자료 파일을 웹 페이지에 업로드합니다.
3.  **학습 모드 선택:** '강의 복습', '벼락치기', '퀴즈 생성' 중 원하는 학습 모드를 선택하고, 필요한 경우 세부 설정을 조정합니다.
4.  **AI 분석 및 결과 확인:** AI가 자료를 분석하여 선택된 모드에 맞는 결과물을 생성하고, 웹 페이지에서 즉시 확인할 수 있습니다.
5.  **결과물 저장 및 관리:** 생성된 학습 결과물은 자동으로 개인 보관함에 저장되어, 필요할 때마다 다시 열람하거나 활용할 수 있습니다.

## 이런 분께 추천해요

*   방대한 강의 자료를 효율적으로 정리하고 핵심 내용을 빠르게 파악하고 싶은 학생
*   강의 내용을 더 깊이 이해하고, 체계적인 복습 자료를 통해 학습 효과를 높이고 싶은 학습자
*   시험 대비를 위해 스스로 문제를 만들거나 구조화된 학습 자료가 필요한 사용자

## 현재 상태 및 향후 계획

### 현재 상태

현재 원페이지 웹 서비스는 초기 핵심 기능 개발을 완료하고 배포 환경에서 기능 검증 및 안정화 단계를 거치고 있습니다. 사용자 피드백을 바탕으로 지속적인 개선을 진행 중입니다.

### 향후 계획

*   **음성 자료 분석 기능 추가:** 강의 음성 파일(MP3, WAV 등)을 업로드하면 AI가 이를 텍스트로 변환하여 분석하고 학습 결과물을 생성하는 기능 도입
*   **다국어 지원 강화:** 더 다양한 언어로 된 강의 자료를 분석하고, 생성된 학습 결과물을 여러 언어로 제공하는 기능 확장
*   **학습 통계 및 진도 관리:** 사용자별 학습 패턴 분석 및 진도율 트래킹 기능을 추가하여 개인화된 학습 가이드 제공
*   **협업 기능:** 스터디 그룹 등 사용자 간 강의 자료 및 학습 결과물 공유, 공동 학습 기능 도입
